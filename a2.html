Arduino code used to control the LEDs:

  

const int buttonPin = 2;  
const int redPin = 11;
const int greenPin = 10;
const int bluePin = 9;

int brightness = 0;
int fadeAmount = 5;
// 0 = red, 1 = green, 2 = blue
int color = 0;  
int buttonState = 0;

void setup() {
  pinMode(buttonPin, INPUT);
  pinMode(redPin, OUTPUT);
  pinMode(greenPin, OUTPUT);
  pinMode(bluePin, OUTPUT);
}

void loop() {
  // This reads whether the button is pressed or not and then stores it to 'buttonState'
buttonState = digitalRead(buttonPin);

// Run the main logic inside a for loop (loops once, so behavior stays the same)
for (int i = 0; i < 1; i++) {

  // If the button is pressed, the led turns on
  if (buttonState == HIGH) {
    // show the selected color with the current brightness
    if (color == 0) {           // red
      analogWrite(redPin, brightness);
      analogWrite(greenPin, 0);
      analogWrite(bluePin, 0);
    } 
    else if (color == 1) {      // green
      analogWrite(redPin, 0);
      analogWrite(greenPin, brightness);
      analogWrite(bluePin, 0);
    } 
    else if (color == 2) {      // blue
      analogWrite(redPin, 0);
      analogWrite(greenPin, 0);
      analogWrite(bluePin, brightness);
    }

    // fade in or out
    brightness += fadeAmount;

    // when fully bright or fully off, reverse direction
    if (brightness <= 0 || brightness >= 255) {
      fadeAmount = -fadeAmount;

      // when fading out is done (brightness == 0), move to next color
      if (brightness <= 0) {
        color++;
        if (color > 2) color = 0;  // loop back to red after blue
      }
    }

    delay(20);
  } 
  else { // turn everything off if button is not pressed
    analogWrite(redPin, 0);
    analogWrite(greenPin, 0);
    analogWrite(bluePin, 0);
    brightness = 0;
    fadeAmount = 5;
    color = 0;
  }
}

3. Measure and record the actual voltage across one of your LEDs when it's on. How does this compare to the theoretical forward voltage for your LED color?
When the LED shows the color red and it is fully on, the reading is about 2.0 V across the LED. Calculations: Voltage Supply: 5.0 V Measured V_LED: 2.0 V Resistor: 220 ohm I = (5.0 - 2.0) / 220 = 3.0 / 220 = 13.6 mA The value 13.6 mA current value matches the theoretical forward voltage range: 1.8 - 2.2 V The measured voltage falls within the expected range, confirming that the LED is operating as specified.
